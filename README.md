# Projeto Iniciante de Análise de dados

### Sobre o projeto

Esse projeto serviu para consolidar habilidades básicas em análise de dados com Python e Pandas. A base de dados é sobre o RH de uma empresa que contém registros de pessoas 'fictícias' de vários departamentos. A base está iniciamente suja, por esse motivo, foi necessário fazer o tratamento dos dados da base para depois fazer uma análise concreta das informações.

### Linguagens, Ferramentas e Bibliotecas Utilizadas
- Python
- Pandas
- Jupyter Notebook
- Vs Code
